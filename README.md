# Hospital-Analytics
End-to-end Hospital Analytics Dashboard using SQL, Excel, Power BI, and Tableau
Project Overview
This Project analyzes hospital booking data to monitor revenue performances, occupancy rate, ADR (Average Daily Rate), RevPAR (Revenue per available room) using SQL, Excel, Power BI and Tableau.
